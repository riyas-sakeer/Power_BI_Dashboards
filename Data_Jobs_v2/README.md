## 📌 Introduction

Navigating the data job market can feel like a maze with information scattered everywhere. This dashboard (V2.0) is designed specifically for **👩‍💻 Job Seekers, 🔄 Job Transitioners, and 🔁 Job Swappers** to cut through the clutter.

Using a real-world dataset of **2024 data science job postings**, this project provides a **streamlined interface** to explore market trends, salary insights, and skill demand across different roles.

## 🛠 Skills Showcased

### ⚙️ Data Transformation (ETL)

Cleaned and prepared raw data using **Power Query** by handling blanks, changing data types, and creating new columns.

### 🧮 Measures & KPIs

Created key metrics such as **Median Yearly Salary**, **Median Hourly Salary**, and **Job Count** using Power BI calculations.

### 📊 Data Visualization

Implemented **Column, Bar, Line, and Area charts** to analyze job trends and salary comparisons.

### 🌍 Geospatial Analysis

Used **Map Charts** to visualize the global distribution of data jobs.

### 📋 KPI Cards & Tables

Displayed high-level insights using **Cards** and detailed information using **Tables**.

## 💡 Additional Skills Demonstrated

🎨 **Dashboard Design** – Designed a clean and intuitive report layout.
🔄 **Power Query ETL** – Performed data cleaning and transformation.
🧩 **Data Modeling** – Built relationships using **Star Schema principles**.
